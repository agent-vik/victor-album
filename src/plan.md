# Victor Album 项目计划

## 项目概述

从博客（victor42.eth.limo）中选取长途游记文章的图片，做成静态网页相册。

- **GitHub 仓库**：agent-vik/victor-album
- **GitHub ID**：265177004
- **部署方式**：GitHub Pages（agent-vik.github.io/victor-album）
- **项目性质**：独立项目（不整合到博客 Hugo 系统中）
- **自定义域名**：后续再加，暂不考虑

---

## 已确认的设计决策

| 决策项 | 结论 |
|--------|------|
| 文章筛选 | 手动指定，配置化 |
| 入选文章 | 仅长途游记（7篇），不含博物馆、国美毕业展、短途见闻 |
| 图片区分 | 不区分，文章内所有图片全部收录 |
| 图片来源 | 直接使用 CDN 原图链接（已压缩，无需生成缩略图） |
| 页面结构 | 首页列表页 + 详情页（每篇文章一个） |
| 列表页展示 | 每篇文章展示标题 + 预览图 |
| 详情页布局 | 瀑布流，图片按文章中原始顺序排列 |
| 图片配文 | 保留图片的 alt 文本作为说明文字 |
| 视觉风格 | 遵循博客现有风格，支持亮色/暗色切换 |
| 字体 | Google Fonts - Lato（与博客一致） |
| 列表页排序 | 按文章发布时间倒序（最新在前） |
| 列表页预览 | 每篇文章展示24张预览图，网格布局（非瀑布流） |
| 详情页图片排序 | 按文章中的原始顺序 |
| 维护方式 | 手动触发（用户告知新文章 → 更新配置 → 执行更新） |

### 文章 URL 规律

- 早期（2022-2024）：`/post/{数字slug}/`（如 `/post/3617/`）
- 后期（2025+）：`/post/{英文slug}/`（如 `/post/trip-to-xi-an/`）
- 每篇中文文章有对应的英文版 `/post-en/{slug}/`

### 图片格式与命名演化

| 时期 | 文件名格式 | 扩展名 | 来源可识别性 |
|------|-----------|--------|-------------|
| 2022年早期 | `IMG_YYYYMMDD_HHMMSS` | .jpg | 手机拍摄（可识别） |
| 2022年早期 | `VID_YYYYMMDD_HHMMSS` | .jpg | 视频帧截图（可识别） |
| 2022年早期 | 长文件名含 `1920x1080` + hash后缀 | .jpg | 网页下载图（可识别） |
| 2023年过渡期 | 32位哈希 | .jpg | 来源不明 |
| 2024年至今 | 32位哈希 | .webp | 来源不可识别 |
| 特殊 | `.gif`（>5MB） | .gif | 手机录屏（可识别） |
| 特殊 | `cover_` 前缀 | .webp | 封面/头图（可识别） |

### HTML 结构

```html
<article>
  <header>...</header>
  <section>
    <!-- 正文内容，图片在 <section> 内 -->
    <img src="..." alt="..." loading="lazy">
  </section>
  <footer>
    <!-- 日期和阅读时长 -->
  </footer>
</article>
```

- 所有图片均使用 `loading="lazy"`
- 图片全部托管在 cdn.victor42.work，无外部图片
- 注意：页面侧边栏（相关文章推荐）中也包含图片，不属于文章正文内容
